# The Machinist Mind

AI-powered personalized learning tool with structured difficulty levels (D1–D5).

## What it does

- Analyzes your learning goal
- Generates structured study material at your chosen difficulty level (D1–D5)
- Creates unique tests to verify your knowledge
- Remembers your progress

## Tech stack

- React + TypeScript (Vite)
- Claude API (Anthropic)
- Custom backend (architecture TBD)
- Custom architecture — no shortcuts, no cookie-cutter solutions

## Philosophy

Every technical decision is intentional. No off-the-shelf backend services, no generic solutions. Built from scratch, the right way.
