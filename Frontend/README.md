# The Machinist Mind — Frontend

React + TypeScript (Vite) frontend.

## Getting started

```bash
npm install
npm run dev
```

Runs on: http://localhost:5173

## Stack

- React 18
- TypeScript
- Vite
- React Router

## Structure

src/

├── components/    # Reusable UI elements

├── features/      # Feature modules (auth, chat, level)

├── assets/        # Images, icons

├── styles/        # Global CSS

├── types/         # TypeScript types

└── utils/         # Helper functions
